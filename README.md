# Amazon Sales Dashboard 📊

This Power BI dashboard analyzes Amazon sales data by:

- 📅 Month and Year
- 🌎 State and City
- 📦 Category and Status
- 💰 Total Sales, Profit, and Orders

## 🛠 Tools Used
- Power BI Desktop
- DAX Measures
- CSV Data Cleaning
